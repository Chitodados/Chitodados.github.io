Monitoramento e Previsão dos Níveis dos Rios

Análise estratégica para otimização do fluxo de materiais

Seu negócio depende de portos fluviais? Com nossa ferramenta, você pode prever os níveis dos rios e planejar suas operações logísticas com mais eficiência.

Funcionalidades Principais

  • Automação de web scraping com Selenium
  • Atualização de dados no GitHub
  • Execução programada com GitHub Actions
  • Dashboard interativo com Streamlit

Estrutura do Repositório

  • webscraping_dash.py: Coleta e atualização dos dados
  • nivel_dos_rios.csv: Base de dados consolidada
  • requirements.txt: Dependências do projeto
  • update_csv.yml: Automação no GitHub Actions
